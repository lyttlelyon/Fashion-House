Included Pages

  • index.html - Home page, featured products, collection band, and appointment callout.
  • products.html - Product showcase with local item images and JavaScript filtering.
  • board.html - Board member cards with a shared profile modal.
  • appointments.html - Inquiry form with event appointment prefill support.
  • events.html - Clickable event cards and JavaScript countdown.
  • dom.html - Document Object Model structure.

Interactive Elements

  • Mobile navigation - Toggles the edge-to-edge menu on small screens.
  • Product filters - Shows matching product cards without changing their order.
  • Board modal - Opens profile details from each board card.
  • Event links - Send visitors to the appointment form with the event context.
  • Countdown - Reads the next major event date from the events page markup.

Asset Pattern

  • assets/ - Stores product, board, hero, appointment, and event imagery locally.
  • styles.css - References local background images for page heroes.
  • HTML img tags - Use local JPG paths with descriptive alt text.
